## 📝 Project Description
This project performs an in-depth Object-Oriented Design (OOD) analysis
of Python's pathlib library. We identify and demonstrate all four OOP
principles — Abstraction, Inheritance, Encapsulation, and Polymorphism
— using real code examples from the library.

We also build a custom Python extension by subclassing pathlib's Path
class to add meaningful new functionality.

## 🔍 OOP Principles Covered
- **Abstraction** — Path() factory hides OS complexity
- **Inheritance** — 3-level chain with no code duplication
- **Encapsulation** — Private attributes + read-only properties
- **Polymorphism** — Same methods, different behaviour per OS
